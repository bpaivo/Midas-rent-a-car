# Midas Rent a Car

Este projeto contém tudo o que você precisa para rodar o aplicativo localmente.

## Como rodar localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure a chave `VITE_GEMINI_API_KEY` (ou similar) no arquivo [.env.local](.env.local).
3. Inicie o aplicativo:
   `npm run dev`
